# @startpage/components

Unstyled react components for your startpage.

## Button

A button element.

<ButtonDemo />

### Props

| name       | type                       | description                                     |
| ---------- | -------------------------- | ----------------------------------------------- |
| active     | boolean \| undefined       | Applies an active css class.                    |
| className  | string \| undefined        | Pass your own css classes for unique styling.   |
| disabled   | boolean \| undefined       | Disables the element.                           |
| leftExtra  | ReactFragment \| undefined | Component which is displayed on the left side.  |
| rightExtra | ReactFragment \| undefined | Component which is displayed on the right side. |
| onClick    | () => void \| undefined    | Callback fired when the element is clicked.     |

### SCSS template

```scss
button.stpg-button {
  > span.stpg-button-extra { }

  &.active { }
  :active { }
  :focus-visible { }
  :hover { }
}
```

## Modal

A popup window for e.g. settings.

<ModalDemo />

### Props

| name            | type                    | description                                       |
| --------------- | ----------------------- | ------------------------------------------------- |
| backdrop        | boolean \| undefined    | Adds an layer between modal and page content.     |
| className       | string \| undefined     | Pass your own css classes for unique styling.     |
| enterTransition | Transition \| undefined | Transitioning when modal enters the screen.       |
| leaveTransition | Transition \| undefined | Transitioning when modal leaves the scree.        |
| open            | boolean \| undefined    | Mounts / unmounts the modal.                      |
| onClose         | () => void              | Callback fired when a closing event is triggered. |

** Transition **

| name       | type   | description                                         |
| ---------- | ------ | --------------------------------------------------- |
| from       | string | CSS class applied before triggering the transition. |
| to         | string | CSS class applied to trigger the transition.        |
| transition | string | CSS class applied during the transition.            |

### SCSS template

```scss
div.stpg-modal {
  > div.stpg-modal-backdrop { }
  > div.stpg-modal-window { }

  /* Pass those into the Transition object */
  &.hidden { }
  &.visible { }
  &.transition { }
}
```

## Select

Element for selecting between several options.

<SelectDemo />

### Props

| name        | type                       | description                                                                                           |
| ----------- | -------------------------- | ----------------------------------------------------------------------------------------------------- |
| className   | string \| undefined        | Pass your own css classes for unique styling.                                                         |
| disabled    | boolean \| undefined       | Disables the element.                                                                                 |
| options     | Option[]                   | Options of the select.                                                                                |
| placeholder | string \| undefined        | Placeholder if no option is selected.<br/> If a placeholder is passed, an empty option will be added. |
| value       | string \| undefined        | Current value of the element.                                                                         |
| leftExtra   | ReactFragment \| undefined | Component which is displayed on the left side.                                                        |
| rightExtra  | ReactFragment \| undefined | Component which is displayed on the right side.                                                       |
| onBlur      | () => void \| undefined    | Callback fired when the element loses focus.                                                         |
| onChange    | (value: string) => void    | Callback fired when the elements value changes.                                                       |
| onClose     | () => void \| undefined    | Callback fired when the content is collapsed.                                                         |
| onFocus     | () => void \| undefined    | Callback fired when the element gains focus.                                                          |
| onOpen      | () => void \| undefined    | Callback fired when the content is expanded.                                                          |

### SCSS template

```scss
div.stpg-select {
  > button.stpg-select-header {
    > span.stpg-select-extra { }
    > span.stpg-select-text {
      &.placeholder { }
    }

    &[aria-expanded="true"] { }
    :focus-visible { }
    :hover { }
  }

  > ul.stpg-select-options {
    > li {
      &.active { }
      &.selected { }
    }
  }
}
```

## Switch

A component to switch between two states.

<SwitchDemo />

### Props

| name       | type                                  | description                                     |
| ---------- | ------------------------------------- | ----------------------------------------------- |
| checked    | boolean \| undefined                  | Checked state of the switch.                    |
| className  | string \| undefined                   | Pass your own css classes for unique styling.   |

  > span.stpg-switch-track { }

  &[aria-checked="true"] { }
  :focus-visible > span.stpg-switch-track { }
}

## TabGroup & Tab

Components to switch between different contents.

<TabDemo />

### Props

** TabGroup **

| name         | type                                 | description                                           |
| ------------ | ------------------------------------ | ----------------------------------------------------- |
| className    | string \| undefined                  | Pass your own css classes for unique styling.         |

      &[aria-selected="true"] { }

      :focus-visible { }
      :hover { }
    }
  }

  > div.stpg-tabgroup-panel { }
}

## Text input

A text input element.

<TextInputDemo />

### Props

| name         | type                                                                                   | description                                                                                                        |
| ------------ | -------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| autocomplete | string \| boolean \| undefined                                                         | Enables HTML basic autocomplete, pass a boolean for on / off or a string with the (HTML) type of the autocomplete. |
| autoFocus    | boolean \| undefined                                                                   | Input will be focused on load if set to true.                                                                       |
| className    | string \| undefined                                                                    | Pass your own css classes for unique styling.                                                                      |
| disabled     | boolean \| undefined                                                                   | Disables the element.                                                                                               |
| placeholder  | string \| undefined                                                                    | Placeholder which is shown if value is undefined.                                                                   |
| type         | "email" \| "number" \| "password" \| "search" \| "tel" \| "text" \| "url" \| undefined | Type of the input element, defaults to "text".                                                                        |
| value        | string \| undefined                                                                    | Current value of the element.                                                                                       |
| leftExtra    | ReactFragment \| undefined                                                             | Component which is displayed on the left side.                                                                      |
| rightExtra   | ReactFragment \| undefined                                                             | Component which is displayed on the right side.                                                                     |
| onBlur       | () => void \| undefined                                                                | Callback fired when the element loses focus.                                                                       |
| onChange     | (value: string) => void                                                                | Callback fired when the elements value changes.                                                                     |
| onFocus      | () => void \| undefined                                                                | Callback fired when the element gains focus.                                                                        |
| onKeyPress   | (key: string) => void \| undefined                                                     | Callback fired when a key is pressed while the input is focused.                                                    |

### SCSS template

```scss
div.stpg-text {
  > span.stpg-text-extra { }
  > input.stpg-text-input {
    ::placeholder { }

    :focus-visible { }
    :hover {}
  }
}
```
